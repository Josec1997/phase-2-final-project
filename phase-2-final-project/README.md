## Project Truck CLub
National Truck Club Page to showcase member's Builds

## How the App Works
Click on top tabs to take you to respected page
Can Delete member's trucks on Gallery or Add new ones on Add Truck form

## Description
This is my personal phase 2 project for flatiron final, Texas edition. I love trucks, that isnt a secret! 
The backend of the project is mocked by unRESTful API , the user then is free to add as many truck members as they like to the gallery. If the we lose a member or gain more, can edit on the specified pages.

## Project Content
Files in project: react/html/css-style/json 
Contains: 8 components/ 4 client side routes/ and POST/GET Request(s)
